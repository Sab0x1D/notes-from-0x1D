# YaraScanTUI
A fast terminal-based YARA scanner with auto-venv setup, rule management, and clean match output â€” ideal for quick triage and testing custom rules.

Detailed markdown guide: `https://sab0x1d.github.io/blogs/gy-tool/`

- Textual-based TUI
- Directory picker (no manual paths needed)
- Cached rule compilation
- Exports: JSON, Markdown, HTML
- One-click launch scripts (Windows/macOS/Linux)
- PyInstaller packaging hint

## Run
  1. install python (download from official page)
  2. add python path by right-click `add-python-to-path.bat` and `run as Administrator`
  3. double-click `run.bat`
